c-ffi-example
======================

Example usage of the [Haskell](https://haskell.org) C FFI.

```bash
~/Desktop/c-ffi-example » nix-build
~/Desktop/c-ffi-example » ./result/bin/person                                                                                         dmj@Davids-MacBook-Pro
Name: david
Age: 30

```
